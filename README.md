# MucleMemory

## サイト概要
*筋トレレビューサイト
### サイトテーマ
*体を鍛えている人による、これから体を鍛える人、今まで鍛えていた人すべてに情報発信する場を提供する。

### テーマを選んだ理由
*筋トレが趣味なのですが、なかなか思うように結果が出ず、悩んでいる部位ごとに効果的な筋トレを知れたらと思ったからです。

### ターゲットユーザ
*筋トレするすべての人

### 主な利用シーン
*筋トレで思うように効果が出なかったときや、もっと効率的に筋トレをしたいとき。
